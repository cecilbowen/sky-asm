# Sky ASM

This repo contains some miscellaneous patches I made for use with the game Pokemon Mystery Dungeon: Explorers of Sky.

##  Usage
These are all ASM patches, so to use them, you'll have to do something like this:
1. Unpack your Explorers of Sky ROM with a tool like [Tinke](https://github.com/pleonex/tinke "Tinke") or [NDSFactory](https://github.com/Luca1991/NDSFactory "NDSFactory")
2. Apply the patch with [armips](https://github.com/Kingcom/armips "armips")
3. Repack the ROM with your patched files

Each patch .asm file has the info on which region it's for (US, EU or JP) as well as what files it's targeting to patch.
